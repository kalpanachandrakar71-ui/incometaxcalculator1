# incometaxcalculator1
Build an Antigravity-based Indian Income Tax Dashboard (FY 2025–26) with real-time calculations, glassmorphism UI, dual-pane layout, and dark/light mode. Compare New vs Old regimes with recommendation engine, sliders (80C, 80D, HRA, NPS), charts, marginal relief, 87A rebate, PDF export, mobile-first design, and CA consultation.
